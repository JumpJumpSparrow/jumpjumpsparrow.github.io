Jekyll 环境的搭建

Posted by Miaocf on August 10, 2020

最近换了台电脑，需要重新安装Jekyll环境，在这里记录下Mac安装Jekyell的过程。

一.环境准备

在安装Jekyll之前，需要具备以下条件：

  1. Xcode 命令行
  2. Homebrew
  3. Ruby

1. Homebrew

什么是 Homebrew ?. 用一句话说明：Homebrew installs the stuff you need that Apple (or your Linux system) didn’t.

Homebrow安装：

1
2

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

将以上内容粘贴到命令窗口，即可安装 Homebrew，中途可能需要输入用户密码。

2. 安装命令行工具

1

xcode-select --install

如果你已经安装了Xcode，无需进行此步。

3.安装Ruby

Jekyll 需要 2.5.0以上版本的 Ruby，macOS Catalina 10.15 自带 ruby 2.6.3，如果你是之前的 mac 系统，需要更新下Ruby。

1
2
3
4
5
6
7

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew install ruby

添加路径
echo 'export PATH="/usr/local/opt/ruby/bin:$PATH"' >> ~/.bash_profile
echo 'export PATH="$HOME/.gem/ruby/X.X.0/bin:$PATH"' >> ~/.bash_profile
‘X.X.0’为本地Ruby版本

4.安装Jekyll

安装 Bundler and Jekyll

1

gem install --user-install bundler jekyll

创建测试

1
2
3
4

jekyll new my-awesome-site
cd my-awesome-site
~/my-awesome-site $ bundle exec jekyll serve
# => Now browse to http://localhost:4000

如果已经创建 Jekyll，可能需要运行 gem update jekyll来更新Jekyll。 然后使用bundle update来更新依赖的包
